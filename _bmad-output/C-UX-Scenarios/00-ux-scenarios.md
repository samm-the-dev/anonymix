## Scenarios

### [01: The Enthusiast's Round](01-the-enthusiasts-round/01-the-enthusiasts-round.md)
**Persona:** The Enthusiast — Discover music through trusted friends
**Pages:** Session Home, Tape List, Tape: Submission Phase (includes song search), Tape: Playlist Ready, Tape: Voting Phase, Tape: Results/Reveal
**User Value:** Complete a full tape cycle — submit, listen via playlist, vote, experience the reveal, walk away with new music
**Business Value:** Core participation loop completed — validates the product concept

---

### [02: The Casual's First Session](02-the-casuals-first-session/02-the-casuals-first-session.md)
**Persona:** The Casual — Effortless participation, fear of too many steps
**Pages:** Join/Invite Landing, Auth/Sign Up (includes platform link). Profile/Account moved to shared/global pages.
**User Value:** Go from invite link to ready-to-play in under 2 minutes, feeling included
**Business Value:** New player onboarded with zero dropoff — participation pool grows

---

### [03: The Host's New Session](03-the-hosts-new-session/03-the-hosts-new-session.md)
**Persona:** The Host — Create rounds effortlessly
**Pages:** Create Tape
**User Value:** Session created with tapes queued, prompts written, invite link shared — under 5 minutes
**Business Value:** New session launched with runway — game loop sustainability

---

### [04: The Host's Session Edits](04-the-hosts-session-edits/04-the-hosts-session-edits.md)
**Persona:** The Host — Fear of admin becoming a chore
**Pages:** Session Admin
**User Value:** Add tapes, rearrange order, manage members — under a minute, still feels like fun
**Business Value:** Session stays active over time, host doesn't burn out
