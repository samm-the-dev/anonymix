# 03: The Host's New Session

**Project:** Anonymix
**Created:** 2026-03-24
**Method:** Whiteport Design Studio (WDS)
**Design Intent:** C (Discuss)
**Design Status:** not-started

---

## Transaction (Q1)

**What this scenario covers:**
Create a new session and set up the first batch of tapes — establish the group container, write compelling prompts, and get rounds queued up so the game can begin.

---

## Business Goal (Q2)

**Goal:** Engagement Quality + Organic Growth
**Objective:** Rounds that run smoothly start with a well-set-up session and thoughtfully prompted tapes. If creating sessions is easy, more people will host.

---

## User & Situation (Q3)

**Persona:** The Host (Tertiary)
**Situation:** The organizer friend who just convinced the group to try Anonymix. They're at their desk after sending the group chat message, riding the momentum of "let's do this" — they want to have the session ready before the group's enthusiasm fades.

---

## Driving Forces (Q4)

**Hope:** Get the session live with a few great prompts queued up before the group's excitement cools.

**Worry:** Setup taking so long or being so confusing that they lose momentum and the whole thing stalls before it starts.

---

## Device & Starting Point (Q5 + Q6)

**Device:** Mobile
**Entry:** Already in the app — just finished setting up their own account, now taps to create a new session.

---

## Best Outcome (Q7)

**User Success:**
Session created, 2-3 tapes queued with prompts they're proud of, invite link copied and dropped in the group chat — total setup under 5 minutes.

**Business Success:**
New session launched with multiple rounds ready — game loop has runway, not just a single tape that might stall.

---

## Shortest Path (Q8)

1. **Create Tape** — Creates the session, writes the first tape prompt, sets deadline, then adds 1-2 more tapes with prompts ✓
